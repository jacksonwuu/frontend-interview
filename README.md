汇总一些前端常见的面试题，大多数没有标准答案， 不同的人可能有不同深度的回答。

同时，这些问题也可以用来检验自己的前端技术水平。（很多问题还是挺考验水平的。）

建议不要背答案，而是尽可能地理解背后的原理，这样才能游刃有余。
